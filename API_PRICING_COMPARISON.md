# API 定价对比：Google Speech-to-Text vs AI Builder Space Audio API

## Google Speech-to-Text API 定价

### Speech-to-Text V2 API（标准识别）

按每月处理的音频分钟数计费，采用分层定价：

| 每月音频分钟数 | 价格（USD/分钟） |
|---------------|-----------------|
| 0 - 500,000 分钟 | **$0.016** / 分钟 |
| 500,000 - 1,000,000 分钟 | **$0.01** / 分钟 |
| 1,000,000 - 2,000,000 分钟 | **$0.008** / 分钟 |
| 2,000,000+ 分钟 | **$0.004** / 分钟 |

### Speech-to-Text V2 API（动态批处理）

- **动态批处理识别**：**$0.003** / 分钟（比标准识别便宜 75%，但需要 24 小时处理时间）

### Speech-to-Text V1 API

| 模式 | 免费额度 | 超出后价格（USD/分钟） |
|------|---------|---------------------|
| 带数据记录 | 60 分钟/月 | **$0.016** / 分钟 |
| 不带数据记录 | 60 分钟/月 | **$0.024** / 分钟 |

### 计费说明

- 按秒计费，向上取整到最接近的秒数
- 多声道音频：每个声道单独计费
- 如果 API 返回响应（包括空响应），音频即被视为已处理并计费
- 服务器错误不计费

### 免费额度

- **V1 API**：每月 60 分钟免费额度
- **V2 API**：无免费额度

## 价格对比总结

### Google Speech-to-Text API

**优势：**
- ✅ 定价透明，公开可查
- ✅ 有免费额度（V1 API：60 分钟/月）
- ✅ 批量处理价格更低（$0.003/分钟）
- ✅ 按使用量付费，无隐藏费用

**价格范围：**
- 标准识别：$0.004 - $0.016 / 分钟（根据使用量）
- 批处理：$0.003 / 分钟

**适用场景：**
- 需要透明定价的项目
- 需要免费额度进行测试
- 可以接受 24 小时延迟的批处理场景

### AI Builder Space Audio API（基于 OpenAI Whisper）

**优势：**
- ✅ 价格较低：$0.006 / 分钟（标准）或 $0.003 / 分钟（Mini）
- ✅ 支持 99+ 种语言
- ✅ 支持说话人识别（Diarization）
- ✅ 基于 ChatGPT 的语音识别技术

**价格：**
- 标准：$0.006 / 分钟
- Mini：$0.003 / 分钟

**注意**：实际价格可能因 AI Builder Space 的服务费而有所不同

## 注意事项

1. **实际成本可能更高**：
   - Google Cloud 的其他服务（如 Cloud Storage、Cloud Functions）可能产生额外费用
   - 生产环境的基础设施成本可能使总成本翻倍或三倍

2. **多声道音频**：
   - Google Speech-to-Text 对每个声道单独计费
   - 例如：30 秒的 4 声道音频 = 120 秒的计费时间

3. **免费额度**：
   - Google V1 API 提供 60 分钟/月的免费额度
   - 适合测试和小规模使用

4. **批处理 vs 实时**：
   - 批处理价格更低，但需要等待时间
   - 实时处理价格较高，但响应更快

## 结论

**对于大多数使用场景：**

1. **最便宜选项**：
   - Google 批处理：$0.003/分钟（需等待24小时）
   - AI Builder Space Mini：$0.003/分钟（实时）

2. **性价比最高（实时处理）**：
   - AI Builder Space（标准）：$0.006/分钟
   - 比 Google 标准（$0.016/分钟）便宜 **62.5%**

3. **适合大规模使用**：
   - Google 标准：使用量超过 200 万分钟/月时，价格降至 $0.004/分钟

4. **适合测试和小规模使用**：
   - Google V1 API：每月 60 分钟免费额度
