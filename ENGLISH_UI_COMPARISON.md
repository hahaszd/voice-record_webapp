## 🎯 **设计原则应用**

### **1. 图标优先** ✅
```
自动复制 → 📋 (clipboard icon)
转录完成提醒 → 🔔 (bell icon)
自动录音 → 🔁 (loop icon)
音频源 → 🎙️ (microphone icon)
转录时长 → ⏱️ (stopwatch icon)
```

**效果**: 减少 ~50% 的文字量，界面更清爽

### **2. 精简英文** ✅
```
语音录制与转录 → Voice Transcription (6字 → 2词)
转录结果 → Transcript (4字 → 1词)
准备就绪 → Ready (4字 → 1词)
正在转录中 → Transcribing (6字 → 1词)
```

**效果**: 平均缩短 60-70%

### **3. Tooltip 补充** ✅
```html
<label title="Auto-copy transcript to clipboard">📋</label>
<label title="Notify when transcription completes">🔔</label>
<label title="Auto-start recording after transcription">🔁</label>
```

**效果**: 鼠标悬停显示完整说明，不影响界面简洁

### **4. 国际通用** ✅
```
30秒 → 30s (seconds的国际缩写)
1分钟 → 1m (minutes的国际缩写)
5分钟 → 5m

🎤 = 全球通用的麦克风图标
🔊 = 全球通用的音响图标
⏱️ = 全球通用的计时器图标
```
