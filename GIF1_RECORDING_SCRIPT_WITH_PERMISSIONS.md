## 💡 常见问题处理

### Q1: 权限弹窗没有出现？
**原因：** 浏览器已记住权限  
**解决：** 
1. 关闭无痕窗口
2. 重新打开无痕模式
3. 确认是新的隐私会话
4. 重新录制

### Q2: 转录失败了？
**原因：** API 问题或说话不清楚  
**解决：**
1. 重新录制
2. 说话更清晰
3. 检查网络连接

### Q3: GIF 文件太大？
**原因：** 分辨率太高或时长太长  
**解决：**
1. 降低质量到 85%
2. 减少颜色到 128
3. 缩小分辨率（1280x720 → 1024x576）
4. 加速等待部分（2x 速度）

### Q4: 鼠标移动不流畅？
**原因：** 移动太快或帧率太低  
**解决：**
1. 重新录制，慢慢移动鼠标
2. 提高帧率到 20 fps（下次录制）
3. 后期删除抖动的帧
