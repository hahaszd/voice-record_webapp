# 🎉 Tally感谢页面设置详细教程

**难度**: 超简单  
**时间**: 2分钟

---

## 📍 在哪里设置感谢页面？

### 方法1: 通过表单底部设置（最简单）⭐

#### 第1步：找到"After submit"设置入口

在Tally表单编辑器里：

1. **向下滚动到表单最底部**
   - 你会看到提交按钮（"Submit" 或你自定义的按钮文字）
   
2. **在提交按钮下方**，找到灰色的文字：
   ```
   After submit > Redirect to URL
   ```
   或者
   ```
   After submit > Show message
   ```
   
3. **点击这一行文字**
   - 会弹出设置面板

---

#### 第2步：选择显示方式

弹出的面板里会看到3个选项：

**选项1: Show message**（显示消息）⭐ **选这个！**
- 用户提交后看到一个感谢消息
- 最简单，最常用

**选项2: Redirect to URL**（跳转到URL）
- 提交后跳转到另一个网页
- 暂时不需要

**选项3: Close popup**（关闭弹窗）
- 只在表单是弹窗形式时用
- 暂时不需要

**👉 点击选择 "Show message"**

---

#### 第3步：编辑感谢消息

选择"Show message"后，会看到编辑区域：

**1. Heading（标题）** - 在这里输入：
```
🎉 You're all set!
```

**2. Message（正文）** - 在这里粘贴：
```
Thanks for joining the VoiceSpark community!

We'll keep you updated on:
• New features and improvements
• Productivity tips and tricks
• Exclusive early access opportunities

In the meantime, start capturing your ideas now! 🚀

---

Having trouble? Got feedback?
Email us: hello@voicespark.com
```

**3. 可选按钮（Button）** - 暂时不添加
- 如果想添加，可以设置：
  - Button text: `Start using VoiceSpark`
  - Button link: `https://your-website.com`

---

#### 第4步：预览效果

1. 点击右上角 **"Preview"** 按钮
2. 在预览模式下：
   - 填写一个测试邮箱
   - 点击提交
   - 查看感谢页面是否显示正确
3. 满意后点击 **"Exit preview"**

---

#### 第5步：保存
- Tally会自动保存你的设置 ✅
- 不需要手动点保存

---

## 📍 方法2: 通过左侧设置菜单（备选）

如果你找不到表单底部的设置，也可以这样做：

1. **看表单编辑器左侧**
   - 找到菜单列表
   
2. **找到并点击 "After submit"**
   - 或者叫 "Ending" / "Thank you page"
   
3. **进入后续同样的设置界面**
   - 选择 "Show message"
   - 编辑标题和正文

## 📱 感谢页面最佳实践

### ✅ 应该包含的元素

1. **感谢用户** 
   - "Thanks for joining!"
   - "You're all set!"
   
2. **确认下一步**
   - "We'll email you when..."
   - "You'll hear from us soon..."
   
3. **行动号召（CTA）**
   - "Start using now!"
   - "Try it out!"
   
4. **联系方式（可选但推荐）**
   - Email地址
   - 或其他联系方式

### ❌ 避免的错误

- ❌ 太长（超过100个字）
- ❌ 没有emoji（显得冷冰冰）
- ❌ 没有告诉用户下一步做什么
- ❌ 用语太正式商务（"Dear user..."）

## 🔍 找不到"After submit"？试试这些

### 可能的原因和解决方法：

**原因1: 滚动不够**
- 解决：继续向下滚动，在提交按钮下方

**原因2: 界面语言不同**
- 可能显示为：
  - "Ending"
  - "Thank you page"
  - "提交后"（如果是中文界面）

**原因3: 在错误的页面**
- 确认你在：表单编辑页面（Form Builder）
- 不是：Dashboard或其他页面

**原因4: Tally界面更新了**
- 新界面可能在左侧菜单
- 找：Settings → After submit

## 💡 Pro Tips

### 1. 使用Emoji让感谢页更友好


```
✅ 好：We'll keep you updated on new features and tips
❌ 差：You are now subscribed
```

### 4. 提供联系方式建立信任
```

❌ 差：[没有联系方式]
```

## ❓ 常见问题

**Q: 可以添加图片吗？**
A: Tally的免费版可能不支持，但emoji已经够用了

**Q: 可以添加按钮吗？**
A: 可以！在Message下方有"Add button"选项

**Q: 感谢页面可以自定义样式吗？**
A: 会继承你设置的主题颜色（橙色）

**Q: 用户可以返回修改邮箱吗？**
A: 提交后不能返回，但可以再次访问表单重新提交
