# 📖 Help文档更新 - 移动端/桌面端音频源说明

**更新时间**: 2026-02-04  
**版本**: v66  
**改动**: 添加移动端/桌面端音频源支持说明

---

## 🎯 更新原因

**用户需求**:
> "我希望把这个说明页也大概解释一下我们现在新更改的，关于在移动端只支持麦克风的说明，只有在电脑端可以使用系统音频的这个部分。"

**背景**:
- v63版本隐藏了移动端的音频源选择按钮
- 移动端只支持麦克风（浏览器限制）
- 需要在帮助文档中向用户解释这个变化
