## ✨ **主要功能**

### 录音模式
- 🎤 麦克风录音
- 🔊 系统音频录制
- 🎵 混合录制（麦克风+系统音频）

### 智能自动化
- ⚡ **自动录音** - 转录完自动开始下一轮
- 📋 **自动复制** - 结果直达剪贴板
- 🔔 **完成提醒** - 浏览器通知不错过

### 转录时长
- 30秒、1分钟、5分钟 随心选择

## 🎯 **亮点特性**

### 1. 无缝录音
转录期间继续录音，**零间隙**，不丢失任何内容

### 2. 隐私保护
所有数据仅保存在浏览器本地，不上传服务器

### 3. 简洁易用
打开网页 → 点击录音 → 点击转录 → 完成！

## 📊 **技术细节**

```
前端: 原生 HTML/CSS/JS（零依赖）
后端: FastAPI + AI 转录 API
部署: Railway（自动化 CI/CD）
性能: 20-30秒转录一分钟音频
```
